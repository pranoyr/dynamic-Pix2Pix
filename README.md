# Dynamic-Pix2Pix 
Implementation of Dynamic-Pix2Pix https://arxiv.org/pdf/2211.08570.pdf

# Getting Started
## Prerequisites
* PyTorch 
* Python 3


## Train
```
python main.py --cfg cfg/exp1.yaml
```


## Inference
```
python inference/inference.py --img_path <image-path> --checkpoint <path-to-checkpoint>
```
